# 変成将棋 コード規約

## 基本方針

- 名詞は日本語、動詞は英語キャメルケースを使う
- 型の種別を接頭辞で明示する

## 命名規則

| 種別 | 接頭辞 | 規則 | 例 |
|------|-------|------|----|
| クラス | `C` | `C` + 日本語名詞 | `C盤面`, `C駒`, `C局面` |
| 構造体 | `S` | `S` + 日本語名詞 | `S手`, `S升座標` |
| インターフェイス | `I` | `I` + 日本語名詞 | `I評価器`, `I探索器` |
| 抽象クラス | `A` | `A` + 日本語名詞 | `A駒`, `A探索器` |
| enum型 | `E` | `E` + 日本語名詞 | `E駒種`, `E手番`, `E成否` |
| enum値 | なし | 日本語 | `E駒種.歩兵`, `E手番.先手` |
| メソッド | なし | 英語動詞（キャメルケース） + 日本語名詞 | `Get合法手()`, `Is王手()`, `Set手番()`, `Evaluate盤面()` |
| 変数・フィールド | なし | 日本語名詞 | `盤面`, `手番`, `持ち駒` |

## コメント

- 日本語で記述する
